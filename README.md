# Contact API

Contact API is a simple API, where a user can get a quick overview over all contacts resources
like person, skills...
Implemented with API-First Approach

## Technological stack

- OpenJDK 14
- Gradle 6.5
- Spring Boot 2.3.0
- Spring MVC
- Spring Data JPA
- Spring Security
- OpenAPI Generator
- Swagger
- Lombok
- MapStruct 1.3.1
- Spock 1.3 for testing
- H2 as embedded Database
- Docker
- Docker Compose

## Installation
```bash
docker-compose up
```

## Usage

### Swagger URL
http://localhost:8080
